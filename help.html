Map Instructions

Meta Controller Instructions

Click on nodes to expand immediate children or collapse all children. Right click to see other actions.
To add a new expert to the tree (After creating one in the "Build An Expert" interface): drag the square in the legend and drop above the node you want to add the expert to, as a leaf.

Build an Expert Instructions

Here you can edit existing experts (accessed through right clicking a node in the meta controller) or develop new experts. If you create an expert, you can export it, or save it to the meta controller.

NOTE: If you reset the expert view, you are also resetting the connection to the meta controller.

Log Instructions

Cover intersection "blind spots" here. Click on a table row to view extended information.
